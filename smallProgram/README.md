# 微信小程序实现移动点餐

### 项目说明：
微信小程序：实现一个点餐 

### 目录结构：
- images — 存放项目图片
- pages — 存放项目页面相关文件
- style — 存放独立wxss样式文件，可import引入
- utils — 存放utils文件，可require引入
- json — 存放配置文件
